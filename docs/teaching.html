Teaching

I also presented a few workshops in different places that generated a fair amount of education material. Here I am slowly compiling some of the resources I’ve produced:

Software and tutorials

Evolved: open software for teaching evolutionary biology

evolved, an open-source R package for undergraduate and graduate teaching in evolutionary biology. It provides tutorials with clear learning objectives focused on data visualization, hypothesis testing, and simulation-based reasoning using real datasets, enabling students to explore both microevolutionary processes (e.g., selection and drift) and macroevolutionary patterns from molecular and fossil data. The software stable version is available on CRAN, the development version can be found in my github. It was published in the Journal of Open Source Education and received the 2026 T. H. Huxley Award for education resources from the Society for the Study of Evolution.

Please reach out if you would like to use evolved. Some of the investigation prompts are intentionally not publicly available online, but I’m happy to share this material with any human interested in using the software to teach evolutionary biology.
